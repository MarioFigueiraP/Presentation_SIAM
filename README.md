# Presentation SIAM


This presentation introduces two approaches—recursive and distributed—to address the challenges posed by the "curse of big data" within the INLA framework. Key topics include methodological and computational hurdles, recursive strategies for sequential analysis as new data arrives, and techniques for partitioning data to enable efficient distributed inference.
